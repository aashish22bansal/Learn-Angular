String Interpolation

data-binding-string-interpolation works!

Say, we do not want to hard code the output here because it would be realistic if the server contains information about a specific server.

For example, we want to create a template as: Server with ID ___ is ___.

So, we can see that we have two dynammic fields here and it would make sense to derive them from the TypeScript Code (which have been hard coded for this example in the file).

We can use Data Binding to obtain the information as: Server with ID {{ serverId }} is {{ serverStatus }}.


Now, say that the data is being returned in a method (from the sample method defined in the TypeScript file).

Server with ID {{ serverId }} is {{ getServerStatus() }}.
