Using ngIf to Output Data Unconditionally

Server was created, server name is {{serverName}}

Now, from the above part, suppose we want to show only a small part stating, "Server was created" and there is no need for the text, "No Server was created" to be displayed. We can show this message conditionally using the ngIf directive. It works like an if statement

Now, we initally have the text to which we will add the directives. Before adding the ngIf Directive, we need to add a * (star) because it is a structural directive which means that it changes the structure of out DOM which means it either adds the element or it does not add that element.

After this, we will setup the condition within the "" (Double Quotes). For ngIf, it has to be any expression returning true or false deciding whether it should be added or not. For this, we will also create the boolean serverCreated variable in the TypeScript file and create the required if() condition(s) and then bind the serverCreated variable to ngIf Directive.
