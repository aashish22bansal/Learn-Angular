There are other type of Directives known as Attribute Directives which are called so because they look like normal HTML Attributes and are used without a *.

Suppose, we want to create the serverStatus (in the TypeScript File) dynamically. Now, we will first create the constructor(){{ '{}' }} in which we will generate a random value between 0 and 1. Based on the value generated, if the value is less than 0.5, then we will set the serverStatus to offline otherwise we will set it to online.

Now, suppose we want to change the background color also based on this value, then for that we can create another Directive and use the ngStyle for this. Also, for it to do something, we will need to provide some configuration to it. For this, we will use Property Binding on this which will get a JavaScript Object where we will find the Key-Value pairs within which Style name is the Key and the Style Value is the Value.
