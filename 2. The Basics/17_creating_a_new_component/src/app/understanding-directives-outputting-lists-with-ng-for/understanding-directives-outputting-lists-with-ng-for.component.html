Outputting Lists with ngFor

We know that we are changing the values for the content present but we are not adding to the list which is being displayed because it is totally static. We can change this and make it a dynamic list with the ngFor Directive.

For this, we will create a new Property in the TypeScript Class as an Array, named servers with some initial content in them. We will push more values into this array within the constructor().

We will place another directive with a * (because this is also a Structural Directive changing the DOM itself) to add the ngFor directive with the following syntax:

*ngFor = "let TEMPORARY_VARIABLE_NAME of PROPERTY_ARRAY_NAME_IN_TYPESCRIPT_FILE"

So, we can now use the server (temporary variable) within the template.
