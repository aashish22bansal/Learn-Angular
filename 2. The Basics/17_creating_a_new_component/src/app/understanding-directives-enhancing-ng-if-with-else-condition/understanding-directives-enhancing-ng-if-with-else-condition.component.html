Now, it is obvious that if we have an if() condition somewhere, then we can have else{{'{}'}} as well. For the above code, we can place a Local Reference to the code and we will have to change the "p" Tag in HTML to ng-template.

The ng-template is a Component/Directive shipping with Angular which we can use to mark places in the DOM and we will add the paragraph ("p" HTML Tag) within this Template with the Content present within it. So, this marks a certain spot in the template which we want to be shown conditionally. Now, to just show it conditionally, we just enhance the *ngIf by adding else and then the Local Reference that we created.
